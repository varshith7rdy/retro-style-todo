## Getting Started

Follow these steps to run the project locally:

```bash
# Clone the repository
git clone https://github.com/varshith7rdy/retro-style-todo.git

# Change to the directory
cd retro-style-todo

# (If using npm/yarn/pnpm, install dependencies)
npm install

# Start the development server
npm start
```

Visit [http://localhost:3000](http://localhost:3000) or the specified port to view the application in your browser.
